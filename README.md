# 🧠 Browser-Based Whiteboard for Online Teaching

Built a simple, fast, and practical whiteboard designed specifically for **live teaching and explanations** — directly in the browser.

---

## 🚀 Overview

I wanted a tool that is:

* Lightweight ⚡
* Easy to use 🎯
* No installation required 🌐

So I built this **browser-based whiteboard from scratch**.

---

## ✨ Key Features

* ✏️ Smooth drawing & natural handwriting
* 📄 Multi-page canvas (no more cluttered boards)
* 🔍 Zoom & pan for easy navigation
* 🧰 Essential tools:

  * Pen
  * Shapes
  * Text
  * Eraser
* 📤 Export as **PDF or Image** in one click
* 🌐 100% browser-based — works instantly

---

## 🎯 Use Cases

This whiteboard is perfect for:

* 👨‍🏫 Teachers & educators for live classes
* 🎓 Students explaining concepts
* 💻 Developers for live coding walkthroughs
* 🎥 YouTubers creating educational content

---

## 🛠️ Tech Stack

* HTML5 Canvas
* Vanilla JavaScript
* CSS (Modern UI + Glass Effect)
